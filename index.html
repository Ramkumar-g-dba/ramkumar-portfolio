What I Built

Project 01 · Dec 2025 – Feb 2026

End-to-End Modern Data Stack Pipeline

AWS RDS (MySQL) → Fivetran → Snowflake → Power BI
🎯 Problem Statement

Build an automated cloud data pipeline that continuously syncs transactional MySQL data into a Snowflake data warehouse and delivers live business insights via Power BI.

  • Architected multi-layer ELT framework (RAW → STAGING → ANALYTICS) in Snowflake
  • Configured Fivetran with CDC (binary log) for incremental sync from AWS RDS MySQL
  • Implemented Star Schema with Fact & Dimension tables for BI optimization
  • Automated pipeline using Snowflake Streams (APPEND_ONLY) + Tasks (5-min schedule)
  • Integrated AWS S3 with Snowflake using IAM Roles and Trust Policies
  • Delivered live Power BI dashboard via DirectQuery with advanced DAX measures
✅ Result

Near real-time pipeline with 5-min refresh · ₹1.78L revenue tracked · 80% orders completed · Live Power BI dashboard

Multi-Source ELT Analytics Pipeline

AWS S3 + Oracle SQL → Informatica IICS → Snowflake → Power BI
